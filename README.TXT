Alunos: Vinicius de Freitas Castro, Matheus Diogo da Silva, Giordanna Santos e Souza.

PROGRAMA: ESSA É UMA SIMULAÇÃO DE IDE EM C, ESSE PROGRAMA FAZ A CHECAGEM DE PONTOS E VIRGULAS, CHAVES, PARENTESES E POSSUI FUNÇÃO DE AUTOCOMPLETAR.

PARA A EXECUÇÃO DESSE PROGRAMA É NECESSÁRIO QUE TENHA INSTALADO A BIBLIOTECA CONIO.H, COLOCANDO OS SEGUINTES COMANDOS NO TERMINAL:

1 - $ sudo apt-get update
2 - $ sudo apt-get upgrade
3 - $ sudo apt-get install git
4 - $ git clone https://github.com/zoelabbb/conio.h.git
5 - $ cd conio.h
6 - $ sudo mv conio.h /usr/include/

7 - FECHAR E ABRIR O VSCODE.

LINK DO TUTORIAL OFICIAL: https://github.com/zoelabbb/conio.h
EM CASO DE DÚVIDAS, TUTORIAL EM VÍDEO: https://www.youtube.com/watch?v=V119IzUgwJw&ab_channel=Avi

PASSO A PASSO DE EXECUÇÃO:

1 - COMPILE VIA TERMINAL UTILIZANDO: $ g++ -g -Wall ide.cpp -o ide
2 - VIA TERMINAL EXECUTE O COMANDO: $ ./ide
3 - DIGITE SEU CÓDIGO EM C
